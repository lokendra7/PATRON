# PATRON
PATRON: Exploring respiratory signal derived from non-contact face videos for face anti-spoofing

Face authentication provides non-contact, user-friendly, covert, and low-cost acquisition. Despite this, face authentication is avoided in safety-critical applications because an adversary can easily spoof it. Several methodologies have been explored in the literature, but all of them, including remote Photoplethysmography(rPPG), are insufficient to detect the 3D face mask. The 3D face mask attack is considered the most potent attack, and it cannot be correctly detected even after consolidating different methodologies. It motivates us to explore a different methodology for face anti-spoofing based on respiration rate because it provides complementary information with the existing methodologies.
